# Hangman Game 🎮

This is a console-based Hangman game written in C++.

## Features
- Random word selection
- Multiple categories (Fruits, Animals, Countries)
- Score tracking system
- File handling for leaderboard
- Interactive gameplay

## How to Run
Compile the code:
g++ hangman.cpp3.cpp -o hangman

Run the program:
./hangman
